First-time setup (~5 min)
  1. Create a new Google Sheet.
  2. Extensions → Apps Script. Paste the contents of apps-script.gs (shipped with this file) into Code.gs and save.
  3. Deploy → New deployment → Type: Web app. Set Execute as: Me and Who has access: Anyone. Authorize.
  4. Copy the web app URL (ends in /exec) and paste it above.
  5. Pick a session code, click Test connection, then Save.
